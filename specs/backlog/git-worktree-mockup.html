git worktree 需要一個實體資料夾。兩種慣例：

選項 1｜repo 旁 sibling 資料夾（推薦）
Documents/
├─ 我的終端機/　　　　　← 主 repo（main）
└─ 我的終端機-worktrees/
　　├─ feat-worktree/　 ← worktree 1
　　└─ fix-statusline/　← worktree 2
  • repo 保持乾淨，不用改 .gitignore
  • 檔案樹 / 搜尋 / watcher 不會掃到彼此
選項 2｜repo 內 .worktrees/
我的終端機/
├─ src/
├─ .worktrees/　　　　← 要加 .gitignore
│　├─ feat-worktree/
│　└─ fix-statusline/
└─ package.json
  • 全部收在一個資料夾裡，好找
  • 主 repo 的搜尋/檔案樹/watcher 要特別排除，否則會看到重複檔案、效能變差
