Polydesk

多工作區開發終端機 — 模仿 VSCode 的檔案總管／原始碼控制／編輯器，左側常駐工作區列表同時開發與監控多專案，每工作區各有獨立終端機與 Claude×Playwright 自動化測試。

基底：vercel 設計系統（Geist · 藍 accent · shadow-as-border）→ 客製成三主題

深色 冷調中性（預設） 淺色 Vercel 乾淨白 暖色 Claude 陶土米色

關鍵畫面（每頁右上可即時切換三主題）

01

主畫面（Shell）

完整 VSCode 風版面：工作區列表 + 活動列 + 檔案總管 + Monaco 分割編輯器 + 整合終端機（claude 接線中）+ 狀態列。
02

原始碼控制（git GUI）

變更清單 + stage/commit + 分支切換 + 並排 diff（紅綠）。呼叫系統 git，完整日常工作流。
03

多工作區監控總覽

差異化核心：所有工作區的 git／程序／未存檔／自動化狀態一覽，背景持續更新，含「無法使用」狀態。

規格涵蓋（v1）

  • 工作區列表 切換 / 背景運行 / 監控
  • 整合終端機 多 shell · real PTY
  • Monaco 編輯器 + 多語言 LSP 自動偵測
  • 完整 git GUI diff/commit/分支/歷史
  • Claude × Playwright 每工作區獨立 profile
  • 全域搜尋 + 分割編輯器
  • 三主題 深 / 淺 / 暖
  • 打包 Windows app + 自動更新

Polydesk（codename，可改）· 凍結前 UI 方向對焦稿 · 設計系統為美學靈感、非官方品牌資產
